DebuggerKit

Bug Tracker

DebuggerKit is a ticketing system that allows you to manage projects and submit tickets for software bugs, defects, issues and much more. This application allows you to develop a workflow between teams and individuals on resovling issues.

Features

  • Create and submit tickets.
  • Assign users to projects.
  • Edit and manage projects and tickets.
  • Time stamped tickets and history for past reference.
  • Built on HTML, CSS, Javascript, jQuery, Bootstrap, C#, SQL Server and ASP.NET Core MVC..

FinancialKit

MVC Financial Portal

Are you looking for a way to manage your finances better? The Financial Kit is a great application that can help you manage your budget and provide tools to help you monitor your spending.

Features

  • Create a budget for yourself and your household.
  • Add and manage banks accounts
  • Edit and review your transactions.
  • Make deposits, withdraws and much more.
  • Built on HTML, CSS, Javascript, jQuery, Bootstrap, C#, SQL Server and ASP.NET Core MVC..

FinancialKit API

Financial Portal API

The FinancialKit API is a web service that you can use to access data from the application. You can use it to retrieve data from the database without a direct connection.

  • This API is built with C#, ASP.NET Core, Npgsql, pgAdmin, Swagger/Swashbuckle.
