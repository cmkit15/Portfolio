Debugger Kit

Bug Tracker

The Debugger Kit is a defect tracking system that is utilized to report software bugs or issues. This application allows you to develop a workflow between teams and individuals on resovling issues.

Features

  • Create and submit tickets for all issues.
  • Assign tickets to a specific department, team or individual.
  • Edit and manage tickets.
  • Time stamped tickets for past and future reference.
  • Built on HTML, CSS, jQuery, Javascript, MVC, C#, SQL Server.

Financial Kit (MVC)

Financial Portal (MVC)

Are you looking for a way to manage your finances better? The Financial Kit is a great application that can help you manage your budget and provide tools to help you monitor your spending.

Features

  • Create a budget for yourself and your household.
  • Add and manage banks accounts
  • Edit and review your transactions.
  • Make deposits, withdraws, transfers and much more.
  • Built on HTML, CSS, jQuery, Javascript, MVC, C#, SQL Server.
